How to Use the LoadingScreen Component with Character Selection

The LoadingScreen component has been enhanced to allow users to select their preferred mascot character.

// Import the component
import { LoadingScreen } from './components';

// Use in your component
function MyComponent() {
  const [loading, setLoading] = useState(true);
  const [selectedMascot, setSelectedMascot] = useState('coin'); // Get from user preferences

  useEffect(() => {
    // Simulate data loading
    setTimeout(() => {
      setLoading(false);
    }, 3000);
  }, []);

  // Handle mascot selection
  const handleMascotChange = (mascotType) => {
    setSelectedMascot(mascotType);
    // Save to user preferences
    saveUserPreference('mascot', mascotType);
  };

  return (
    <>
      <LoadingScreen
        show={loading}
        message="Preparing your financial dashboard..."
        theme="primary"
        mascotType={selectedMascot}
        minDisplayTime={2000}
      />

      {!loading && (
        <div>
          <MascotSelector
            selected={selectedMascot}
            onChange={handleMascotChange}
          />
          <div>Your component content here</div>
        </div>
      )}
    </>
  );
}

LoadingScreen Properties

  • show (boolean): Whether to display the loading screen
  • message (string, optional): Custom loading message (default: "Loading your financial future...")
  • theme (string, optional): Color theme - "primary", "success", "warning", or "danger" (default: "primary")
  • mascotType (string, optional): Type of mascot to display - "coin", "piggy", "chart", "rocket", "guru", or "bull-bear" (default: "coin")
  • minDisplayTime (number, optional): Minimum time in milliseconds to display the loading screen (default: 1200)
